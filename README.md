# CS2 Case Pricer

 ## Features
 - Track all base in game cases
 - Real-time Steam market prices
 - Calculate total case value
 - Stores inventory for future price checks

## Installation

 1. Clone the repository
```bash
git clone <link>
cd <repo_name>
```
 2. Install dependencies
 ```bash
 pip install -r requirements.txt
 ```
 3. Run the program
```bash
python main.py
```
## Requirements
- Python 3.13.5
- requests 2.32.4
- colorama 0.4.6
